## 🚀 Features

- **Hero Section** — Introduction with name, tagline, and social links
- **About** — Bio with key stats (CGPA, projects, certifications)
- **Skills** — Grouped technical skills with styled tags
- **Experience** — AIML Internship at QSkill
- **Projects** — 3 featured AI/ML & Web projects with tech stacks
- **Education** — Academic timeline with scores
- **Certifications** — 6 certifications from Cisco, IBM, HackerRank & more
- **Contact** — Email, phone, LinkedIn, GitHub — all clickable
- **Scroll Animations** — Smooth fade-in on scroll
- **Fully Responsive** — Works on mobile, tablet, and desktop

## 🌍 How to Host on GitHub Pages (Free)

1. Create a new GitHub repository named `portfolio`
2. Upload `shivani_portfolio.html` and rename it to `index.html`
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch and click **Save**
5. Your site will be live at `https://shivani-cse.github.io/portfolio/` in a few minutes
